
1 . How do I create a hidden file or folder? How do I display it in the CLI?
   A ; 
   1. To create hidden folder put a "." Before the name of the folder e.g.  -> Mkdir .week1
      To creat hidden file same methods  e.g. -> Touch .homework.text

   2.  To show the hidden file : ls -al 

   3. To create multiple folder: the syntax should be like in this in Mac : mkdir file1 file2 file3;
      Or mkdir file{1,2,3,4};
